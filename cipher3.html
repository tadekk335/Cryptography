Gronsfeld cipher was created by Jos√© de Bronckhorst (earl of Gronsfeld) around 1744. As a Belgian diplomate, a message security during exchange was very important for him. The Gronsfeld cipher is a variation of the Vigenere cipher in which a key number is used instead of a keyword, e.g., 14965. To encrypt using this cipher, a letter of text is taken one by one, and shifted by the corresponding number in the key.The key is repeated to match the message length. For example:

Plain text : NAT

Key : 123

Ciphered text : OCW

This code does not offer any protection. However it is more harder to break that Caesar Cipher.
